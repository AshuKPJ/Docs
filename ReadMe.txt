MLEngine folder contains the following :

1. Source code folder : 
				contains the source code for MlEngine
2. WrapperServicesSourceCode folder : 
				Contains source code for wrapper services on top of MlEngine
				for reference 
3. ML Engine_Demo_v2.pdf :
				Presentation one can scan to understand the thought behind 
				MLEngine . Creation Credit : Praveen 
4. MLEngineDocumentation.docx :
				documents detailed understanding of MLEngine. including
				type of web service, Input/Output , configuration details
5. ClassDiagram.vsd :
				class diagram design 
		
6. Setting Up Tomcat Server.docx :
				can be refered for setting up tomcat server on some server 
				machine
